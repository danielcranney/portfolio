# Ben Weston - Personal Site

## Project description

A personal site for Ben Weston.

## Installation

**Install dependencies**

```bash
npm install
```

**Run development server**

```bash
npm run dev
```

## Packages

- [Next.js](https://nextjs.org/docs)
- [Tailwindcss](https://tailwindcss.com/docs)
